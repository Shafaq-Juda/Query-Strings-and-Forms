For the above form:

The form will be submitted to:

/process_form?fruit=kiwi&color=green

The query string is:

?fruit=kiwi&color=green

(If the user enter kiwi in the first input box and green in the second input box)

Sinatra and Rails read this query string and each name/value pair into params Hash as:

{"fruit" => "kiwi", "color" => "green"
